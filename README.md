## 关于博客

该博客是参考了https://github.com/dongweiming/lyanna 该项目的前端代码,原项目是python技术栈实现的
最近在用Golang写东西，所以通过Golang的Gin框架实现了该项目，从中也学到了很多知识，同时对go开发web也有一个基本的认识，个人觉得
博客项目还是非常适合初学者。


现在该博客已经正式上线https://www.syncd.cn/，但是第一版的很多地方并没有优化，后续会继续进行优化代码.
